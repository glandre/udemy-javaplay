Requirements:

JDK
Typesafe Activator
IDE or Editor, e.g.: IntelliJ IDEA, Eclipse, Netbeans or Sublime


----

Typesafe Activator:
- Download it at: https://www.playframework.com/download
- extract the zip file to some folder, e.g., /home/dev/activator

- add this folder to the path:
$ nano ~/.bashrc

following line to the bottom of the file:
export PATH=/home/dev/activator:$PATH

- to test the activator type:
$ activator ui
